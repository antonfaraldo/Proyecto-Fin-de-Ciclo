📖 ¿Cómo se juega? Lógica para Principiantes

El Buscaminas es un **desafío de pura lógica**. El objetivo es despejar todas las casillas que no contienen minas.

3
La regla de oro: El número que aparece en una casilla indica cuántas minas hay escondidas en las 8 casillas que la rodean (arriba, abajo, izquierda, derecha y diagonales).

Si ves un "1", significa que entre las casillas vecinas solo hay una bomba. Usa tu ingenio para deducir cuál es y márcala con una bandera 🚩.

🎮 Controles y Mecánicas Avanzadas

  • Click Izquierdo: Revela una casilla. Si tocas una mina, la partida termina inmediatamente.
  • Click Derecho: Coloca o retira una bandera 🚩 para marcar minas sospechosas.
  • Chording (Doble Click / Combinado): Si un número tiene ya el número correcto de banderas, se revelarán automáticamente las casillas restantes.
    💀 Riesgo de Error: Si las banderas están mal colocadas, al hacer chording se revelará la mina real y perderás la partida al instante.
  • Iluminación de Vecindad: Al mantener pulsado un número sin haber colocado todas sus banderas, las 8 casillas de alrededor se iluminarán para ayudarte a planificar.

⌨️ Atajos de Teclado (Modo Pro)

  • Tecla [ R ]: Reiniciar partida.
  • Tecla [ P ]: Pausar el juego (oculta el tablero para evitar trampas).
  • Tecla [ ESC ]: Volver al menú principal.

⏱ Contrarreloj NUEVO

Aquí no solo luchas contra las minas, sino contra el reloj.

Reglas del modo:
  • Empiezas con un contador de 60 segundos.
  • Por cada casilla revelada con éxito, sumas +1 segundo extra.
  • Si ejecutas un Chording exitoso, sumas el tiempo proporcional a las casillas abiertas.
  • Si el tiempo llega a 0, las minas detonarán automáticamente.
